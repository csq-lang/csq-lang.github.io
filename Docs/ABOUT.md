_C squared_, _C&sup2;_ or _C to the power of 2_ is a VM-based C superset made with care by [C developers](https://github.com/orgs/csq-lang/people) with one exclusive purpose: being a much more comfortable to use
version of beloved but painful C.

C&sup2; is completely non-profit FOSS (_Free and Open Source Software_), we will never force anything down our users throat (not that we really can being a programming language), and it can be modified and redistributed freely.

Learn more about it in [GitHub](https://github.com/csq-lang/csquared). And, please, **leave your star** if you want to support us!
